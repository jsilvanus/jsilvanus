## Tools for the Church

I'm a priest and a nerd. 

For years, I've been doing small coding projects, but with the increase of *vibe coding* and tools like **Claude** I am finally able to use my few spare hours to make something ready! 

My passion is making small (and sometimes bigger) tools that are actually helpful in my work and the work of others in my field (church life). 

If you're interested in participating on some projects that would be helpful for the *Kingdom of God* and especially the **Finnish Evangelic-Lutheran Church**, get in touch. Reach me via email at Gmail.
